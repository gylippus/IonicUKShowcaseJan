Zourist

How did you find the experience using Ionic?

Amazing, as is the case with everyone else it was easy to make an app even though I came from web dev background. Documentation and forum are amazing as well.

Zourist

Is there any capability or challenge you had to overcome?

FB login integration was a challenge, I tried a plugin(oauth.io) previously which would ask the user to login even if he is already logged in through FB device. We scrapped it and used wizcorps plugin which is awesome.

Zourist

Any advice for other Ionic Developers?

Before implementing anything look for existing ionic apps which have similar feature and have good performance, look into its code(ionic related design aspects) and start from there. It can save a lot of time.
